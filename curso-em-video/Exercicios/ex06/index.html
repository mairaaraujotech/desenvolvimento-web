Principais formatações

Negrito Destaque

Nesta frase temos um termo em negrito usando a tag <b></b> (sem semântica).

Nesta frase temos um termo em destaque usando a tag <strong></strong> (com semântica).

Itálico / Ênfase

Nesta frase temos um termo itálico usando a tag <i></i> (sem semântica).

Nesta frase temos um termo com ênfase usando a tag <em></em> (com semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark></mark>.

Podemos criar um outro texto marcado usando a tag <mark></mark> e mudar a cor usando CSS dentro da tag.

Podemos criar vários textos marcados usando a tag <mark></mark> e mudar a cor usando CSS na tag <style></style> dentro do head da página.

Texto Pequeno

Estamos criando um texto pequeno nesse parágrafo usando a tag <small></small>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado (não semântica).

Texto sobrescrito

Para escrever coisas do tipo x2+3.

Texto subscrito

Para escrever coisas do tipo H2O.
