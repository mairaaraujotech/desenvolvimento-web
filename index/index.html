Índice do Portifólio

Este portifólio é formado pelas atividades executadas durante os cursos que realizei e por ideias testadas para consolidar conhecimento: clones de sites já existentes e um tutorial para estudo. As linguagem utilizadas foram: HTML, CSS e JavaScript.

CURSOS

  • Alura
  • Curso em Vídeo
  • DevMedia

PROJETOS

  • Clones de Sites
  • Tutorial Fron-End
  • Caderno de Anotações
  • JavaScript

Parte do conteúdo (imagens e vídeos) e alguns recursos (fontes) estão hospedados em sites de terceiros com a finalidade de diminiur a quantidade de dados no servidor. Porém, a ocorrência de alteraçoes nos servidores de terceiros pode impactar no funcionamento de nossas páginas.
