Vídeo 04

  • Harmonia de cores
  • Círculo Cromático
  • Representação de aproximadamente 65 milhões de cores
  • Simetria também causa bom impacto visual
  • Cores Primárias: Azul / Amarelo / Vermelho
  • Cores Secundárias: Verde / Laranja / Violeta
  • Cores Terciárias: misturas das cores primárias e secundárias. Sempre leva o nome da cor primária no início, seguido pelo nome da cor secundária. Exemplo: amarelo-esverdeado
  • Cores Frias e Cores Quentes
  • Cores Complementares (extremo oposto, paleta de 2 cores)
  • Cores Análogas (cores imediatamente vizinhas, paleta de 3 cores)
  • Cores Análogas e uma Complementar (paleta de 4 cores)
  • Cores Análogas Relacionadas (cor principal + vizinha + pula uma cor, paleta de 3 cores)
  • Cores intercaladas
  • Cores Triádicas (cor principal + pula três + pula três)
  • Cores em Quadrado (cor principal + pula duas + pula duas)
  • Cores Tetrádicas (formar um retângulo)
  • Monocromia: uma única cor com variações de saturação e luminosidade
  • Paleta de Cores: sempre parte de uma cor principal (que será a cor predominante) depois se escolhe as cores de contraste. Possui de 3 a 5 cores (desconsiderando branco e preto).

- Anatomia dos Tipos - A letra (minúscula) é a base para as outras letras do alfabeto - Uma fonte tem: altura x / altura das maiúsculas / ascendente (haste) / cauda / arco / filete / esporão / terminal / vértice - Corpo é total da altura do conjunto de fonte - Serifa: traço presente em algumas fontes - Letra: glifo. Conjunto de glifos: fonte - Família tipográfica: light + normal + seminegrito + negrito + extranegrito - Categoria de fontes: serifada / sans-serif / monoespaçada / handwriting / fontes comemorativas

- Padrão para tamanhos de fonte - Medidas Absolutas: cm / mm / in / px / pt / pc - Medidas Relativas: em / ex / rem / vw / vh / % - in = polegada - pt = ponto - pc = paica - em = tamanho relativo da fonte - rem = fonte configurada no body (r = root) - ex = tamanho relativo à altura x da fonte - O navegador usa como tamanho padrão de fonte 16px (= 1 em) - Usar sempre o em e px para garantir responsividade

Vídeo 23

- Modelos de Caixas: esse modelo é aplicado tanto para o HTML quando para o CSS (aninhamento) - Conte - Border: contorno em volta do conteúdo, dentro do elemento - Outline: contorno/traçado, fora do elemento - Padding: o espaço entre o conteúdo e a borda - Margin: espaço externo - Cores no console do Chrome (styles): Conteúdo: azul Padding: verde Border: amarelo Margin: laranjado - Box-level: ocupa a largura total da tela e sempre pula pra linha de baixo quando cria um novo elemento div h1-h6 main header nav article aside footer form video - Inline-level: não quebra linha, insere o elemento ao lado do parágrafo, por exemplo, e continua o texto em seguinda span a code small strong em sup sub label button input select
